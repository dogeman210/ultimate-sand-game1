Ultimate Sandbox Game
An interactive, browser-based falling sand physics simulation game. Experience particles interacting in real-time right in your web browser.

Background & Development
This project was developed over a multiple day long streak, logging 50.8 hours of development time to reach Stage 2 status on the Macondo platform.

Building this game was an extensive process, especially when establishing the sand physics. The particles initially scattered erratically or became trapped in unnatural patterns due to logic and syntax errors in the collision routines. Fixing one bug often broke the simulation in a new way.

I utilized A.I tools to help debug these syntax errors and explain the underlying logic issues when the game physics were not working properly. After understanding the required fixes, I manually implemented the updated code and spent hours tweaking the acceleration and friction variables by trial and error until the sand felt completely right and natural when it piles up. I also used A.I to help me write this as I am weak in English.

Installation
Since this game is built entirely with vanilla web technologies, you do not need to install any external packages. You only need a modern web browser like Chrome, Firefox, Edge, or Safari.   

Clone the repository:

Bash
git clone https://github.com/dogeman210/ultimate-sand-game1.git
Navigate to the project folder:

Bash
cd ultimate-sand-game1
Open index.html in your web browser to start playing.

Usage
Use your mouse to draw and drop sand or other elements onto the canvas. Experiment with different particle interactions and watch them settle organically.
