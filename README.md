# ultimate-sand-game1

An interactive, browser-based falling sand physics simulation game. Experience particles interacting in real-time right in your web browser.

## Features
- Real-Time Physics: Watch sand and other elements fall and settle organically.
- Interactive Sandbox: Draw, clear, and experiment with different particle interactions.
- Zero Installation: Runs entirely in the browser without any setup.

## Tech Stack
- Frontend: HTML5 (100%)
- Styling: CSS3 (Embedded)
- Logic: JavaScript (Embedded)

## How to Play Locally

Since this game is built entirely with vanilla web technologies, you do not need to install any external packages.

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```

2. Navigate to the project folder:
   ```bash
   cd ultimate-sand-game1
   ```

3. Open the game:
   Open the `index.html` file in any modern web browser (such as Chrome, Firefox, Edge, or Safari) to start playing.
